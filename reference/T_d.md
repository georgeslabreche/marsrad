# Number of Mars daylight hours

Calculates the duration of daylight (time between sunrise and sunset)
for a given location and season on Mars. Implements Equation 10 from
Appelbaum & Flood (1990).

## Usage

``` r
T_d(Ls, phi)
```

## Arguments

- Ls:

  Areocentric longitude \[deg\]

- phi:

  Planetary latitude \[deg\]

## Value

Number of daylight hours \[h\]
